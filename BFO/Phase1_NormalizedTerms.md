# Normalized Folk Value Terms (Phase 1.2)

This document contains the normalized and de-duplicated list of candidate folk value terms, derived from `Phase1_RawTerms.md`. Terms have been consolidated, and synonyms have been noted. The list is also categorized to distinguish core value concepts from related concepts like goals, capabilities, or states.

This list will serve as the primary input for Task 1.3 (Clustering).

## Related Concepts (Likely Not Primary Values)

These terms are more likely to be goals, capabilities, states of being, or outcomes of realizing a value, rather than values themselves. They are kept here for reference during the modeling process.

- **Capabilities/Skills:** Accuracy, Capability, Competence, Effectiveness, Efficiency, Expertise, Intelligence, Intellect, Knowledge, Performance, Precision, Quality, Rationality, Realism, Rigor, Skill, Usefulness, Utility
- **Outcomes/Goals:** Credibility, Legacy, Popularity, Prosperity, Recreation, Refinement, Significance, Success, Surprise, Vitality, Wealth, Wholeness
- **States of Being:** Consciousness, Contentment, Energy, Grace, Happiness, Health, Intensity, Poise, Silence, Solitude, Wonder, Well-being
